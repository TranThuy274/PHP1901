CSS là viết tắt của Cascading Style Sheets

CSS mô tả các thành phần html được hiển thị trên màn hình và trên các thiết bị khác

CSS tiết kiệm rất nhiều thời gian làm việc

Nó có thể kiểm soát bố cục của nhiều trang web

CSS được thêm vào theo 3 cách: inline, internal(sử dụng style trong the head), external(su dụng file .css ở ngoài)
