HTML ID

Id chỉ ra 1 sự duy nhất không trùng lặp không có cái thứ hai

Giá trị của id có thể được sử dụng bởi css hay js để thực hiện một nhiệm vụ duy nhất cho 1 thành phần với một giá trị cụ thể

Trong css để chọn 1 id duy nhất của thẻ html ta sử dụng selector là #id
